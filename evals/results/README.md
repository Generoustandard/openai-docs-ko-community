# Results 폴더

이 폴더는 `run_eval.py` 실행 결과를 저장하기 위한 자리입니다. MVP에서는 결과 포맷을 아주 단순하게 유지합니다.

## 저장 원칙

- 결과는 JSON 파일로 저장합니다.
- 각 항목에는 두 개의 cosine similarity 점수를 함께 기록합니다.
- 점수 외의 해석 메모는 필요하면 별도 문서나 PR 설명에 남깁니다.

## 예시 파일 이름

- `responses-sample.json`
- `golden-comparison-2026-03-19.json`

## 왜 results/를 미리 두나

- 기여자가 동일한 구조로 결과를 남기기 쉽습니다.
- MVP 단계에서도 평가 흐름이 어디까지 이어지는지 한눈에 보입니다.
- 나중에 CI나 리포트로 확장하더라도 폴더 구조를 크게 바꿀 필요가 없습니다.
